# drawJuliaSet
以ARM Assembly 模擬JuliaSet動畫

使用MAIN(.c)分別呼叫NAME(.s)跟ID(.s)及DrawJuliaSet(.s)函數，計算並決定Frame 二維陣列裡每個元素的值，並以此來決定該元素投影至畫面上的Color。
